# 📊 SQL Data Analysis Project: Data Analyst Job Market Analysis

## Introduction

This project focuses on analyzing the Data Analyst job market using SQL. The primary objective is to extract meaningful insights regarding high-paying jobs, in-demand skills, and the most valuable skill combinations for aspiring and professional data analysts.

By utilizing SQL queries and analytical techniques, this project demonstrates how relational databases can be leveraged to answer real-world business questions and support data-driven decision-making.

---

## Background

The demand for data analysts has increased significantly across industries, creating a need to understand market trends, salary distributions, and skill requirements. This project explores job posting datasets to identify:

- The highest-paying data analyst positions
- Skills required for top-paying jobs
- Most in-demand technical skills
- Skills associated with higher salaries
- Optimal skills based on both demand and compensation

The project was designed to strengthen practical SQL skills through exploratory data analysis and business intelligence techniques.

## Analysis

### 1. Top Paying Jobs Analysis

The first analysis identifies the highest-paying data analyst positions available in the dataset. The query filters jobs based on salary information and ranks them according to annual compensation.

**Objectives:**

- Identify lucrative data analyst opportunities
- Analyze salary distribution
- Understand employer compensation trends

---

### 2. Top Paying Job Skills Analysis

This analysis explores the technical skills required for the highest-paying data analyst roles.

**Objectives:**

- Determine which skills employers value most
- Analyze the relationship between salary and skill requirements
- Identify premium technical competencies

---

### 3. Top Demanded Skills Analysis

This section identifies the most frequently requested skills across data analyst job postings.

**Objectives:**

- Measure market demand for technical skills
- Identify industry-standard technologies
- Understand current hiring trends

---

### 4. Top Paying Skills Analysis

This analysis evaluates which technical skills command the highest salaries.

**Objectives:**

- Determine the most financially rewarding skills
- Analyze salary trends by technology
- Identify high-value career specialization areas

---

### 5. Optimal Skills Analysis

The final analysis combines both demand and salary metrics to identify the most valuable skills for career growth.

**Objectives:**

- Find skills with strong market demand
- Identify skills associated with higher compensation
- Determine optimal learning priorities for aspiring analysts
